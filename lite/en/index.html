Code sample

It's a solve of codewars kata Data Reverse. Time complexity O(n) because it swaps only n/2 items and doesn't another. Space complexity O(1), so it uses only one space for the swap. It reverses in place, there isn't a hidden time or space complexity.

const dataReverse = data => {
  let left, right, i, t;
  for (left = 0, right = data.length - 8; left < right; left += 8, right -= 8)
    for (i = 0; i < 8; ++i) {
      t = data[left + i]; 
      data[left + i] = data[right + i];
      data[right + i] = t;
    }
  return data;
};
